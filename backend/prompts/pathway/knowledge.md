# Germany Care Path Program — Knowledge Base

## Audience
Kenyan healthcare professionals: Nurses (KRCHN/KRN/KRM), Certified Nursing Assistants (CNAs), Clinical Officers, and Health Support Workers seeking employment in Germany.

## Eligibility — Education
- German Language certificate
- All qualifications must be from recognized institutions

## Eligibility — Experience
- No prior experience required but an added advantage if PRESENT

## German language
- A1/A2 to start the program and begin language training
- B1/B2 required before relocation to Germany

## Program timeline (approximate)
1. Preparation (1–2 months): application, credential assessment
23. Placement (1–2 months): school matching, contract, visa processing
4. Relocation (1 month): pre-departure briefing and settlement support in Germany
